# Student Record Management System using Core Java

A console-based Java application to manage student records using Core Java concepts and Object-Oriented Programming principles.

## Features
- Add Student
- View Students
- Search Student
- Menu-driven system

## Concepts Used
- Classes & Objects
- Encapsulation
- ArrayList
- Loops & Methods
- CRUD Operations

## Tech Stack
- Java
- OOPs
- Collections Framework
